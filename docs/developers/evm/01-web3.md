### eth_protocolVersion
___
Returns the current findora protocol version.
#### Parameters
none
#### Returns 
`String` - The current ethereum protocol version
#### Example

Request
```shell
curl localhost:8545 \
  -X POST \
  -H "Content-Type: application/json" \
  --data '{"method":"eth_protocolVersion","params":[],"id":1,"jsonrpc":"2.0"}'
```

Response 
```shell
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": "54"
}
```
